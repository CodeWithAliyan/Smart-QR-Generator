# 🧠 The Instant QR Hub

**Generate. Scan. Connect. Instantly.**  
A modern and responsive QR Code Generator built for speed, elegance, and simplicity. Create fully customisable QR codes with a live preview, multiple dot styles, and PNG/SVG export — all in your browser.

---

## ✨ Features

- ⚡ **Instant QR Generation** – Create QR codes within seconds as you type
- 🎨 **Fully Customisable Design** – Choose dot styles, corner shapes, and custom colours
- 🔄 **Live Preview** – See your QR code update in real-time
- 🧩 **Responsive UI** – Smooth layout across all devices (mobile, tablet, desktop)
- 🧠 **Client-Side Secure** – 100% private processing in your browser, no server involved
- 🖼️ **Modern Dark Theme** – Elegant colour palette: `#1f2833`, `#c5c6c7`, `#66fcf1`
- 📥 **Multiple Export Formats** – Download as PNG or SVG for any use case
- 🎯 **No Sign-Up Required** – Completely free, forever

---

## 🖥️ Tech Stack

- **HTML5** – Semantic structure
- **Tailwind CSS** – Utility-first styling for rapid UI development
- **CSS3** – Custom animations, gradients, and responsive design
- **JavaScript (ES6+)** – Core application logic
- **qr-code-styling** – Advanced QR code generation with customisation options
- **Lucide Icons** – Beautiful, consistent icon set
- **Google Fonts** – Syne (display) + DM Sans (body) for typography

---

## 🚀 How to Use the Web App

1. **Enter Your Content**  
   Type or paste any URL, text, phone number, or email into the input field.

2. **Customise the Design**  
   - Pick a **dot style** (Square, Dots, Rounded, Classy, Classy+, X-Round)
   - Choose a **corner shape** (Square, Dot, Rounded)
   - Adjust **dot colour** and **background colour** using the colour pickers

3. **Preview Live**  
   Your QR code updates automatically as you type or change settings.

4. **Download & Share**  
   - Click **Download PNG** for print or social media
   - Click **Download SVG** for scalable vector graphics in design tools

> 💡 All processing happens in your browser — your data never leaves your device.
